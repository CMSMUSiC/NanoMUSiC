pxl::SoftRelations class

Contents

  • Reference
    • Public types
    • Public functions

This class provides the storage of relations to any Serializable derivatives, by means of the unique Id (UUID) of each object. Each relation has a certain name. In contrast to the standard Relations class, relations between objects with different (or no) object owners can be established. The relations are stored as pairs of names (string) and UUIDs (Id). More than one soft relation with the same name is allowed to be established. Since a relation is not stored as a pointer, it can only be retrieved indirectly. This also means that soft relations exist beyond the lifetime of the related objects, unless they are explicitly removed. An object of type SoftRelation can be added to any class which is supposed to provide such relations, e.g. all derivatives of the Relative such as the Particle.
