Enum documentation

enum pxl::ChunkReader::fileMode

The fileMode flag says if the read-in istream is seekable, or not. It needs to be modified by implementing classes.

enum pxl::ChunkReader::infoMode

The infoMode can be passed as a flag to indicate that a string condition must be fulfilled to read a block, an event, or an information chunk

enum pxl::ChunkReader::statusFlag

The status flag can take one out four values, either before an event (pre-header), or before a block, then either within an information chunk, or an event, or something unknown
