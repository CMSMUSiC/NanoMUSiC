pxl::Basic3Vector class

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions

This class provides a simple threevector with basic algebra. The methods provided are self-explanatory. Note that theta is defined as 0 in z-direction, as usual in collider physics.
