pxl::UserRecords class

Contents

This class is intented to aggregate information complementary to data members in form of string-variant pairs. All PXL physics objects own user records and provide methods for quick access to individual user record entries.

template<typename datatype>
void pxl::UserRecords::change(const std::string& key, datatype item)

Checks if a user record entry identified by key is present, and changes it to the passed value in case it is present. If not, an exception is thrown.

const Variant& pxl::UserRecords::get(const std::string& key) const

Searches and returns the user record item indetified by key; a pxl::Exception is thrown in case the key is not found.

template<typename datatype>
bool pxl::UserRecords::get(const std::string& key, datatype& item) const

Checks if user record entry identified by key is present. If yes, its value is put into the passed item.
