template<class objecttype>
objecttype* pxl::BasicContainer::findObject(const std::string& key) const

This method searches the index for the key and returns a dynamically casted / C++ pointer of type objecttype* to the corresponding object; / in case the key is not found a null pointer is returned.

Serializable* pxl::BasicContainer::getById(Id id) const

Returns a Serializable pointer for a contained object with the passed ID. In case the Serializable is not in the container, 0 is returned.

const std::vector<Serializable*>& pxl::BasicContainer::getObjects() const

template<class objecttype>

const Id& pxl::BasicContainer::getTypeId() const virtual

Id of the object's class (== type). This method must be reimplemented by all derived classes to return a distinct UUID identifying the class, e.g. the Particle.
