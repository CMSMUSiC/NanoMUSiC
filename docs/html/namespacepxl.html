typedef Filter<Particle, ParticlePtComparator> pxl::ParticleFilter

This typedef provides a filter for PXL particles which sorts the particle by transverse momentum. The concrete filter criterion is passed at evaluation time, e.g. the ParticlePtCriterion or the ParticlePtEtaNameCriterion. See the example for further hints on how to use the filters.
